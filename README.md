# Dotfiles

This repo contain my dotfiles and configs, setting up following [this video](https://www.youtube.com/watch?v=y6XCebnB9gs)

## Depentencies

make sure to insall the following:

```zsh
brew install git
brew install stow
```

## installation

```zsh
git clone git@github.com:chenxin-yan/dotfiles.git
cd dotfiles
$ stow .
```
